Para probar este apartado se debe: 

1- Ejecutar el archivo servidor.py con el comando python servidor.py y este se mantendrá a la espera 
    de la conexión por parte de la otra parte que sería el cliente

2- Se ejecuta el archivo cliente.py con el comando python cliente.py y este enviará un mensaje al servidor 
    siguiendo el funcionamiento de la tecnología pull y el servidor le responde con un mensaje. 

Mensaje enviado: Este es el mensaje.  Debe ser respondido.
Mensaje recibido por el cliente: RXN0ZSBlcyBlbCBtZW5zYWplLiAgRGViZSBzZXIgcmVzcG9uZGlkby4=

No se asemejan los mensjes por errores de deserializacion.
